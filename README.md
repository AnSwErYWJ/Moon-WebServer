# Moon——月亮服务器
## Description
最近在使用apache，所以参考了一些资料，用C语言写了一个支持PHP的Web服务器。因为完成时临近是中秋节，所以取名为Moon.  

## Feature
- 支持PHP语言
- 实现GET方法

## Compile and run
```
make
sudo ./Moon
#打开浏览器，在地址栏输入127.0.0.1:8080
make clean
```
## Commit
```
chmod +x git.sh //when first use
./git.sh filename
```
